ps_set_border_dash -- 注記の枠線の、破線の状態を設定する

  bool ps_set_border_dash(resource psdoc, float black, float white)~

|ps_add_weblink| や |ps_add_pdflink|
などの関数で追加されたリンクは、 pdf に変換して pdf
ビューアで見た際には矩形で囲まれて表示されます。
この矩形は、postscript ドキュメントでは表示されません。
この関数は、破線の「線」の部分と「空白」の部分の長さを設定します。

{psdoc} |ps_new| が返す、postscript ファイルのリソース ID。

{black} 破線の長さ。

{white} 破線と破線の間隔。

成功した場合に TRUE を、失敗した場合に FALSE
を返します。
