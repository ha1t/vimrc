ncurses_newwin -- 新規ウインドウを作成する

  resource ncurses_newwin(int rows, int cols, int y, int x)~

この関数は、 実験的 なものです。この関数の動作・
名前・その他ドキュメントに書かれている事項は、予告なく、将来的な
PHP のリリースにおいて変更される可能性があります。
この関数は自己責任で使用してください。

描画要素を入れるための新しいウィンドウを作成します。

端末の大きさはそれぞれ異なりいろいろな値をとる可能性があるので、
ウィンドウを追加する際には |ncurses_getmaxyx|
を使用して使用可能な領域を調べることを忘れないようにしましょう。

{rows} 行の数。

{cols} 列の数。

{y} 原点の y 座標。

{x} 原点の x 座標。

新しいウィンドウのリソース ID を返します。
