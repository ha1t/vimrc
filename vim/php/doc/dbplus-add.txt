dbplus_add -- リレーションにタプルを追加する

  int dbplus_add(resource relation, array tuple)~

タプルをリレーション {relation} に追加します。

{relation}

{tuple} 属性/値の組の配列であり、 指定した {relation}
に挿入されます。

実行に成功すると {tuple}
配列には新規に生成されたタプルの
完全なデータが含まれ、そこにはシーケンスのような暗黙の内に設定
される全てのドメインが含まれます。

この関数は、成功した場合に DBPLUS_ERR_NOERR、
失敗した場合に db++ エラーコードを返します。

この関数は、 実験的 なものです。この関数の動作・
名前・その他ドキュメントに書かれている事項は、予告なく、将来的な
PHP のリリースにおいて変更される可能性があります。
この関数は自己責任で使用してください。
