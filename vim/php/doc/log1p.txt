log1p -- 値がゼロに近い時にでも精度を保つ方法で計算した
log(1 + number) を返す

  float log1p(float number)~

この関数は、 実験的 なものです。この関数の動作・
名前・その他ドキュメントに書かれている事項は、予告なく、将来的な
PHP のリリースにおいて変更される可能性があります。
この関数は自己責任で使用してください。

|log1p| は、 log(1 + {number}) の値を返します。 {number}
がゼロに近い場合でも正確な値が
計算できる方法を使用します。 |log|
は、このような場合には 精度の問題で log(1)
の値を返してしまいます。

この関数は Windows 環境にはまだ実装されていません。

{number} 処理する引数。

log(1 + {number}) を返します。

|expm1| |log| |log10|
